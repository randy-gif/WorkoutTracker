# WorkoutTracker 🏋️‍♂️📱

[![Platform](https://img.shields.io/badge/Platform-Android-3DDC84.svg?style=flat)](https://www.android.com/)
[![Language](https://img.shields.io/badge/Language-Kotlin-7F52FF.svg?style=flat)](https://kotlinlang.org/)
[![UI Toolkit](https://img.shields.io/badge/UI-Jetpack_Compose-4285F4.svg?style=flat)](https://developer.android.com/jetpack/compose)

A custom Android application built from the ground up to track strength training, manage workout routines, and calculate volume over time. Designed specifically for focused training styles like Push/Pull/Legs and hypertrophy tracking.

## ✨ Features

* **Custom Workout Management:** Create and organize custom workout routines with specific exercises.
* **Live Session Tracking:** Track sets, reps, and active exercise sessions in real-time.
* **Volume Calculation:** Automatically calculate workout volume to track progressive overload over time.
* **Detailed History:** A dedicated history screen to review past workouts, analyze progress, and monitor consistency.
* **Dynamic Theming:** Implements a dynamic global theme for a seamless and modern user experience.
* **Persistent Mini-Player:** Keep track of active session timers or ongoing workouts without interrupting navigation.

## 🛠 Tech Stack

* **Language:** [Kotlin](https://kotlinlang.org/)
* **UI Toolkit:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
* **Architecture:** MVVM (Model-View-ViewModel) utilizing state hoisting and unidirectional data flow.
* **IDE:** Android Studio

## 🚀 Getting Started

### Prerequisites
* Android Studio (Latest stable version recommended)
* An Android device running Android 8.0 (API 26) or higher, or an Android Emulator.

### Installation

1.  Clone the repository to your local machine:
    ```bash
    git clone https://github.com/randy-gif/WorkoutTracker.git
    ```
2.  Open **Android Studio**.
3.  Select **Open an existing Project** and navigate to the cloned `WorkoutTracker` directory.
4.  Allow Gradle to sync the project dependencies.
5.  Click the **Run** button (green play icon) to build and install the app on your connected device or emulator.
